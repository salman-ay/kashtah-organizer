# كشتة (Kashtah Organizer) ⛺

A simple, elegant web application to help you prepare and organize your camping trip checklist. Built with vanilla HTML, CSS, and JavaScript.

## Features

- **Multiple Camp Types**: Choose from desert camping (كشتة بر), beach camping (كشتة بحر), or chalet/stay (شاليه/استراحة)
- **Flexible Meal Options**: 
  - Cooking meals (مع طبخ ونفخ)
  - Prepared coffee setup (بدون طبخ - تحضير قهوة)
  - Ready-made drinks (بدون طبخ - جاهز)
- **Customizable Lists**: Edit, add, or remove items from your checklist
- **Save & Load**: Save multiple trip lists and load them later
- **Share**: Copy your list to WhatsApp with formatted text
- **Responsive Design**: Works seamlessly on mobile and desktop devices
- **RTL Support**: Full Arabic language support with right-to-left layout

## How to Use

1. Open `index.html` in your web browser
2. Click "➕ جهّز كشتة جديدة" to start creating a new checklist
3. Select your camp type (desert, beach, or chalet)
4. Choose your meal/drink style
5. Enter the number of people
6. Review and customize your generated list
7. Save your list or copy it to share via WhatsApp

## Technology Stack

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling with CSS variables and modern layouts
- **Vanilla JavaScript**: No frameworks or dependencies
- **LocalStorage**: Save lists locally in your browser

## Browser Support

Works on all modern browsers that support:
- ES6 JavaScript
- CSS Grid and Flexbox
- LocalStorage API
- Clipboard API
